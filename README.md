# AI/ML & Web Scraping Data Entry Internship Assignment

## Objective
Scrape university and course data using Python and organize it in a structured Excel file.

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas
- OpenPyXL

## Structure

### Sheet 1: Universities
Contains university_id, name, country, city, and official website.

### Sheet 2: Courses
Contains course_id, university_id (linked), course details, duration, fees, and eligibility.

## Relational Integrity
Each course is linked to its university using university_id.

- web_scraping_script.py
- university_course_data.xlsx
